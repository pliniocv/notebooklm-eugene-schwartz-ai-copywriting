# Reflection

The quality of NotebookLM responses improved significantly when prompts became more specific, contextualized, and focused on practical applications rather than generic theoretical explanations.

One of the most valuable lessons from this project was learning how prompt design directly influences the depth, relevance, and usefulness of AI-generated responses.
